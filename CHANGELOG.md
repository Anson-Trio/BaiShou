# 更新日志 (Changelog)

所有对本项目的显著更改都将记录在本文件中。

## [1.1.0] - 2026-02-22

### ✨ 新特性
- **AI 架构重构**：采用工厂模式重新设计 AI 客户端，原生支持 OpenAI、Gemini (v1beta) 以及 Anthropic 供应。
- **全新的同步方案**：新增对 **WebDAV** 和 **S3 (包含腾讯云 COS)** 的支持，备份配置现已支持跨设备迁移。
- **数据管理回归**：在「常规设置」中恢复了独立的数据导出、导入、快照恢复与局域网传输入口。
- **响应式架构**：重构主界面布局，完美适配移动端（底部导航）与桌面端（侧边栏）的不同交互逻辑。

### 🛡️ 数据与安全
- **凭据保护**：优化导入逻辑，确保同步时不会因备份文件空值覆盖本地已有的 API Key/URL 凭据。
- **快照保护**：重新实现了覆盖式还原前的自动快照备份逻辑。

### 💄 UI/UX 优化
- **界面精简**：移除桌面端占位的「近期洞察」侧边栏，为日记时间轴提供更广阔的视觉空间。
- **视觉统一**：定义了全新的应用配色体系（Sakura/Akatsuki），优化了全局提示 (AppToast) 与动画效果。
- **设置入口**：优化了移动端设置页的导航流程，解决了返回逻辑中的“循环嵌套”问题。

### 🔧 修复与改进
- **时区修正**：修复了 S3 和 WebDAV 备份记录时间显示为 UTC 的问题，现已统一为本地时区。
- **性能飞跃**：优化 Markdown 渲染逻辑，大幅降低长篇日记列表的滑动卡顿。
- **稳定性**：修复了 Android Release 构建时的混淆报错及局域网传输中的部分设备识别问题。
